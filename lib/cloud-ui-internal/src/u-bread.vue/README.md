# Breadcrumbs

## Example
### Basic form

``` html
<u-bread link="/test" name="Details"></u-bread>
```

# API
### Props/Attrs

| Prop/Attr | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| link | String | `'undefined'` | URL link |
| name | String | `'undefined'` | Current page name |
